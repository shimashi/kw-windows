# Tony's Scripts for app installation and windows update

- setupApps.cmd will install and update chocolatey, zoom, 7zip, Audacity, firefox, gimp, handbrake, inkscape, krita, libreoffice, adobereader, tux-paint, vlc, googlechrome and novabench
- winupdate.cmd will install windows update
> Right click on the file and "Run as administrator"

 # Adam's Powershell Scripts for app installation and windows update

- You can download the full instructions for using the PowerShell installation scripts by clicking on the path

> PowerShellScriptsAdam/Adams_PowerShell_Script_Instructions.docx

And clicking the "Download Raw File arrow.

 - The full command to paste into your Powershell terminal to obtain the latest copy of these files is here:

> curl.exe -L “https://github.com/shimashi/kw-windows/archive/refs/heads/main.zip” -o “$env:USERPROFILE\Desktop\InstallScripts.zip”
